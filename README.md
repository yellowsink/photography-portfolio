# Hazel's Photography Portfolio Backend

A minimal backend for managing my photography, featuring:
 - film rolls
 - photos
 - names and descriptions
 - taken dates
 - "favourites"
 - categories
 - featured categories

The frontend code that talks to with this is not located here, but in [yellowsink/website](https://github.com/yellowsink/website).
